Message: Explain the difference between undefined and null in JavaScript.

java script

Username: tharun reddy

Message: Undefined: Represents a variable that has been declared but has not been assigned a value. It's the default value of uninitialized variables. It's a primitive data type in JavaScript.

Null: Represents an intentional absence of any object value. It's often used to represent that a variable or object does not exist, or has no value. It's a primitive value but is treated as an object.
